========================================
Collection Group Queries
========================================

Para recuperar contenido del usuario a través de múltiples spots:

// Obtener todas las reviews del usuario
collectionGroup('reviews').where('userId', '==', userId)

// Obtener todas las discussions del usuario
collectionGroup('discussions').where('userId', '==', userId)

// Obtener todos los comments del usuario
collectionGroup('comments').where('userId', '==', userId)

NOTA: Requiere crear índices en Firebase Console para collectionGroup queries.

NOTA ADICIONAL: En la práctica, los documentos de `comments` ahora incluyen campos explícitos `contextId`, `sourceType` y `sourceId` (además de `sourceRef` opcional) para facilitar queries específicas por usuario y la extracción de contexto sin depender únicamente de la ruta del documento. Si se usa `collectionGroup('comments')`, se puede priorizar los campos explícitos; en caso de que no estén disponibles, extraer `contextId`/`sourceType`/`sourceId` a partir del `doc.ref.path`.
